# Ambiente di sviluppo locale
{: #local_dev}

Se vuoi configurare un ambiente di sviluppo locale Db2, puoi utilizzare [IBM Db2 Warehouse Developer-C for Non-Production ](https://store.docker.com/images/ibm-db2-warehouse-dev){:external} disponibile dal Docker Store.
{: shortdesc}

Per i prerequisiti e le istruzioni di installazione, seleziona il link per il tuo sistema operativo: 

- [Windows e Macintosh](https://www.ibm.com/support/knowledgecenter/en/SS6NHC/com.ibm.swg.im.dashdb.doc/admin/local_prereqs-Winmac_using_Linux.html){:external}
- [Linux](https://www.ibm.com/support/knowledgecenter/en/SS6NHC/com.ibm.swg.im.dashdb.doc/admin/local_prereqs-Linux.html){:external}
