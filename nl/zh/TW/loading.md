# 載入資料
{: #load}

您可以從位於本端網路的定界格式資料檔案（例如 CSV 或 TXT）、物件儲存庫（Amazon S3 或 IBM Cloud Object Storage（先前稱為 SoftLayer Swift））或 Db2® 伺服器來載入資料。您也可以直接從 Cloudant® 資料庫將資料移入資料庫實例，或是從例如 InfoSphere® DataStage® 的應用程式執行載入處理程序。
{: shortdesc}
